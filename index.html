LN / Extrator de Dados XML

A ferramenta definitiva para automatizar a extração de dados de suas notas fiscais.

Baixar Versão 1.0.1

Requisitos Mínimos:

• Sistema Operacional: Windows 7 com Service Pack 1 (SP1) ou superior.

• Processador: 1 GHz ou superior (qualquer processador dual-core moderno serve).

• Memória RAM: 2 GB.

• Espaço em Disco: 100 MB de espaço livre (para a instalação do programa e arquivos temporários).

• Software Adicional: .NET Framework 4.7.2 ou superior.
